RGB-RGBA

RGB é a abreviatura de um sistema de cores aditivas em que o Vermelho, o Verde e o Azul são combinados de várias formas de modo a reproduzir um largo espectro cromático

RGBA é o sistema de cores formado pelas cores vermelho, verde, azul e pelo canal alfa. O sistema permite exibir todas as cores do sistema RGB e a utilização da transparência de imagem, artifício amplamente usado em softwares de edição de imagem com camadas. Os arquivos de imagem PNG utilizam o sistema RGBA

Cores hexadecimal

As cores em hexadecimal são amplamente usadas em CSS para definir cores de elementos nas páginas da web. Cada cor é representada por um código de 6 dígitos, precedido pelo símbolo #. Este código é dividido em três partes, representando os valores de vermelho (Red), verde (Green) e azul (Blue), ou seja, o formato é #RRGGBB.
